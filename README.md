# C#-Discord-Bot
A C# Discord Bot made with Discord.NET Library

# Features

Proper timer database. If bot off or crash, the timer will still be going!

Many activities you can do.

Rock Paper Scissors between 2 users.

PVP (kinda like that) between 2 users.

Tic Tac Toe (You can play VS AI or with another user.)

To check more, type "c help"
